ggplot(zonas) +

Vemos que hay zonas que estás dispersas. La verdad, al no ser experto en SIGs no se me ocurre otra forma mejor que obtener los centroides y filtrar a mano.


zonas <- zonas %>% 
  mutate(centro = sf::st_centroid(geometry))

centros_coords <- zonas %>% 
  pull(centro) %>% 
  sf::st_coordinates() 

zonas <- zonas %>% 
  mutate(
    x = centros_coords[, 1],
    y = centros_coords[, 2]
  )

zonas <- zonas %>% 
  filter(y <= -33.30) %>%
  filter(y >= -33.70) %>%
  filter(x >= -70.85) %>%
  filter(x <= -70.40)

ggplot(zonas) +
  geom_sf(size = .2, fill = NA, colour = "gray80")

Muy bien, a mano pudimos filtrar y mostrar las zonas que nos interesan para replicar lo que queremos.
